# Exp-C0re
* A super easy example codes that works together.. this will show you how [emu-c0re](https://github.com/egycnq/EmuC0re) ui hijack works and draw UI. So everyone could learn how it works.
* this does not have the Nes EMU in it, but it shows exactly how to build UI, and also how use it to make cool stuff and other EMU's since it helps you to learn how to render and stuff.
* Works on 1920 x 1080 (HD)

# How to build?
* Fork the repo, and edit whatever you need and run the yml. after you get tthr bin, ignore elf. take bin and use my Bin to Lua code (or your own) to cover it to hex, after that put replace local sc = "HEX HERE" and send thr lua via sender to thr game.
